# Step 7: Recalculation Rules

Recalculation must trigger when:

- Measurement changes
- Quantity changes
- Rate changes
- Discount changes
- Advance changes

Recalculation Order:

1. Recompute entry total
2. Recompute material subtotal
3. Recompute grand total
4. Recompute balance

Completion Criteria:

- No stale totals
- Deterministic recalculation flow
